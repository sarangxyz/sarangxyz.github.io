procdump

 Posted on November 12, 2019

to install procdump as default postmortem debugger on windows run command:

procdump -ma -i c:\crashdumps

Here, it will generate minimal dump files with threads and handles -ma in directory c:\crashdumps

to uninstall procdump run following command:

procdump -u
